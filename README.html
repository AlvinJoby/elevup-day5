⛅ ElevUp Day 5 Project

SkyCheck

A clean, mobile-first weather app that gives you real-time conditions, a 6-day forecast, and a global city explorer — all powered by the OpenWeatherMap API.

Live App GitHub Repo

Features

Auto Location

Detects your device's GPS coordinates and instantly loads local weather on launch.

6-Day Forecast

Horizontally scrollable forecast cards showing daily temperature and conditions.

City Search

Search any city worldwide. Supports city, state, country format for precision.

World Explorer

Browse weather for multiple cities at once, and add your own to the board.

Visibility Info

Search results now include visibility distance alongside wind, pressure, and humidity.

Mobile-First

Responsive layout built for phones with a fixed bottom nav bar for easy access.

Technologies Used

HTML5 CSS3 Vanilla JavaScript OpenWeatherMap API Font Awesome 6 Google Fonts (Rubik)

API Endpoints Used

Purpose Endpoint
Reverse geocoding (lat/lon → city name) geo/1.0/reverse
Current weather by city data/2.5/weather
5-day / 3-hour forecast data/2.5/forecast

Getting Started

1

Clone the repository

Open your terminal and run:

git clone https://github.com/alvinjoby/elevup-day5.git
cd elevup-day5
2

Get a free API key

Sign up at OpenWeatherMap and copy your API key from the dashboard. Free tier is sufficient for all features.

3

Add your API key

Open js/main.js, js/search.js, and js/world.js. Replace the placeholder at the top of each file:

let apiKey = "YOUR_API_KEY_HERE";
4

Open in browser

Open index.html directly in any modern browser. When prompted, allow location access so the app can detect your city automatically.

How to Use

Home Tab

Opens to your current location's weather automatically. Shows temperature, humidity, feels-like, and a 6-day scrollable forecast.

Search Tab

Type a city name and press Enter. For ambiguous cities, use the format London, England, GB.

Explore Tab

Preloaded with global cities. Tap the ＋ button to search and add any city to your board.
